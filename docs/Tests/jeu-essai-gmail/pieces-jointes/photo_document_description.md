# Description de la photo de document de test

Cette pièce jointe correspond à une **photo fictive d'un document papier**.

## Description visuelle attendue

- feuille A4 blanche posée sur une table claire ;
- texte administratif imprimé noir ;
- légère inclinaison de la photo ;
- lumière naturelle correcte ;
- bas de page un peu flou pour simuler une capture imparfaite ;
- nom visible en haut à gauche ;
- signature manuscrite en bas de page.

## Usage du test

Cette description permet de :

- créer plus tard une image de test ;
- simuler un document photographié envoyé par un usager ;
- tester les réponses demandant si la pièce jointe est lisible ;
- tester les traitements de mails avec image jointe.

Document de test sans valeur réelle.
