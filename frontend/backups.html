Sauvegardes

Cette page privée permet de lancer un snapshot standardisé du projet, de restaurer un projet BIS et de suivre l’historique utile.

Note produit : cette page est un outil d’administration local sur Mac. Lors de la migration vers le serveur de production, elle devra être séparée de l’espace utilisateur.

Méthode

Règle standard

Le projet actif ne contient plus ses sauvegardes. Les snapshots locaux vont dans une bibliothèque centrale, puis sont repris par Mac, Spark et Extreme Pro.

Action 1

Créer un snapshot projet

Aucun lancement pour le moment.

Action 2

Restaurer un projet BIS

Recréer automatiquement un projet de contrôle à partir d’un snapshot, sans toucher au projet principal.

Aucune restauration lancée pour le moment.
