Administration

Création de règles

Atelier assisté pour transformer une ou plusieurs demandes en règle métier, compléter les paramètres manquants, puis l’enregistrer dans le référentiel.

Processus

Déroulement

Vérification des droits administrateur...

  • Étape 1 : générer une règle candidate à partir des demandes reçues.
  • Étape 2 : laisser l’IA demander les précisions minimales à l’administrateur.
  • Étape 3 : finaliser la règle et l’enregistrer dans le référentiel métier.
  • La règle finale doit viser une réponse one shot ou une demande de complément très ciblée.

Paramètres

Génération IA

Le commentaire opérateur est injecté dans le prompt pour cadrer la création de la règle.

Prompts

Bibliothèque de prompts

Ces prompts servent si tu veux faire travailler un LLM externe, puis réinjecter manuellement son résultat dans ce processus.

Les prompts sont fournis en lecture seule pour servir de référence et de copie rapide.

Usage

Quand utiliser quoi

  • `Créer une règle` : partir d'une ou plusieurs demandes et produire un brouillon structuré.
  • `Finaliser une règle` : reprendre le brouillon avec les réponses admin pour produire la version stockable.
  • `Tester une règle` : vérifier qu'une réponse mail a bien appliqué la logique attendue.
