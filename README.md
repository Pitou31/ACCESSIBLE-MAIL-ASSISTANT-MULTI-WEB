# Accessible Email Assistant

Application d’assistance e-mail accessible.

## Stack
- Frontend: HTML / CSS / JavaScript pur
- Backend: Node.js
- Contrats: JSON

## MVP
- import mail par copier/coller
- analyse IA
- génération de réponse
- modification manuelle
- simulation d’envoi
- saisie audio par transcription locale `faster-whisper`

## Base V2 V3
- politique d'automatisation navigateur par boîte connectée
- sessions d'automatisation navigateur tracées en base
- API backend pour préparer l'intégration de Browser Use
- garde-fous actifs : validation humaine et envoi automatique désactivé par défaut
